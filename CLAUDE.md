# プロジェクト: fate-decoder

## 概要
占術（算命学、マヤ暦、数秘術、西洋占星術、宿曜）を統合した運命解読Webアプリケーション。

## 基本ルール
- コードの変更後は必ず `git add . → git commit -m "簡潔な日本語メッセージ" → git push origin main` を行うこと
- pushが完了したらデプロイURLを報告すること
- コミットメッセージは日本語で簡潔に書くこと
- 既存のデザインやレイアウトを大幅に変更する場合は、事前に確認を取ること

## デプロイ
- ホスティング: Vercel（GitHub連携による自動デプロイ）
- mainブランチへのpushで本番環境に自動反映
- Vercelプロジェクト名: fate-decoder
- GitHubリポジトリ: rinafuku-gif/fate-decoder

## デザイン方針
- 「天体×和」の美学（celestial Japanese aesthetic）
- アニメーション付き星空背景、金のアクセント
- 神秘的かつ洗練されたUI

## コーディング規約
- 日本語コンテンツが中心のため、フォントやレイアウトは日本語表示を優先すること
- レスポンシブ対応必須（モバイルファースト）
- 占術の計算ロジックは正確性を最優先すること

## 注意事項
- 占術の算出ロジックを変更する場合は必ず確認を取ること
- 外部APIキーやシークレット情報はコードに直接書かないこと
